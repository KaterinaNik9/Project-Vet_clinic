# Project: Vet Clinic

Этот репозиторий содержит консольное приложение на C++, моделирующее работу ветеринарной клиники: учет животных, их медицинских записей, назначений (кормление, лечение) и процедур.

---

## 1. Цель разработки

Создать приложение, демонстрирующее основные принципы ООП:

- Инкапсуляция  
- Наследование  
- Полиморфизм  
- Абстракция  

---

## 2. Общая архитектура

Проект построен на иерархии классов с использованием абстрактного базового класса `Animal`.

Все данные управляются контейнером `AnimalHospital`.

### Основные компоненты:

- **Иерархия животных**
  - `Animal` (абстрактный класс)
  - Производные классы:
    - `Dog`
    - `Cat`
    - `Bird`
    - `Reptile`
    - `ExoticAnimal`

- **Медицинские записи**
  - `MedicalRecord`
  - Композиция с `Animal`

- **Лечебные процедуры**
  - `Treatment` (абстрактный класс)
  - Производные:
    - `Vaccination`
    - `Surgery`
    - `Examination`
    - `Medication`

- **Запись на прием**
  - `Appointment`
  - Связывает животное, врача, дату и тип процедуры

- **Контейнер**
  - `AnimalHospital`
  - Хранит коллекцию животных (указатели на `Animal`)
  - Управляет записями и поиском

## 3. Основной функционал

### 3.1 Управление животными

- Добавление нового животного (`new`)
- Удаление по ID или имени
- Просмотр списка
- Поиск по критериям:
  - Кличка
  - Вид / порода
  - Дата поступления

---

### 3.2 Медицинская история

- Добавление записи (диагноз, дата, лечение)
- Просмотр истории животного

---

### 3.3 Назначения и процедуры

- Запись на прием (`Appointment`)
- Назначение лечения (через полиморфизм `Treatment`)
- Вывод:
  - графика кормления
  - протокола лечения (чисто виртуальные функции)

---

### 3.4 Отчеты

- Список животных с текущим статусом
- Статистика по видам
- Список предстоящих процедур
